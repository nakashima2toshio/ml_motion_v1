# 🎥 解析画面 操作マニュアル

**画面**: 左ナビ「解析」（`app/views/analyze.py`）／ 既定で最初に開く画面。

## 目的
mp4 動画を YOLO11 で物体検出し、必要に応じて**セグメンテーション・トラッキング（ID付与）・ゾーン解析（滞留/侵入）**を行い、
注釈付き動画・統計・検出テーブルを表示、CSV/JSON でエクスポートする。

## 事前準備
- 解析する mp4（`.mp4` / `.mov` / `.avi`）。手元に無ければ取得スクリプト：
  ```zsh
  ./scripts/fetch_sample_videos.sh      # data/ に人・車のサンプル
  ```
- （NL要約を使う場合のみ）`.env` に `ANTHROPIC_API_KEY`。

## 画面の構成

### サイドバー（設定）
| セクション | 項目 | 種類 | 既定 | 説明 |
|---|---|---|---|---|
| 入力ソース | ソース | ラジオ | mp4 | 「iPhone…」はリアルタイム画面(P3)用。ここは mp4 を選択 |
| タスク | セグメンテーション（YOLO11-seg） | チェック | OFF | ON でマスク描画（モデル一覧が `-seg` に切替） |
| タスク | トラッキング（ByteTrack / ID付与） | チェック | ON | ID・軌跡を付与 |
| タスク | ゾーン解析（滞留・侵入） | チェック | OFF | **トラッキングON時のみ**有効 |
| モデル | YOLO11 モデル | 選択 | yolo11s | n（軽/速）〜 m（重/精）。seg 時は `-seg` |
| モデル | 信頼度しきい値 | スライダ | 0.25 | 低いほど拾うが誤検出増 |
| 対象クラス | 全クラス（COCO 80） | チェック | OFF | ON で全クラス検出 |
| 対象クラス | クラス（COCO 代表） | 複数選択 | 全代表 | person/bicycle/car/motorcycle/bus/truck |
| 処理設定 | フレーム間引き（N フレームに1回） | スライダ | 1 | 大きくすると速いが粗くなる |
| 処理設定 | 軌跡の長さ（フレーム） | スライダ | 30 | 追跡ON時のみ |
| ゾーン定義 | polygon（JSON） | テキスト | 既定ゾーンA | ゾーン解析ON時のみ表示。**正規化座標 0〜1** |

### メインエリア
- **mp4 をアップロード**（ファイル選択）
- **▶ Run 解析**（アップロード後に押下可）
- 実行後：注釈付き動画プレビュー ／ 右に「結果ペイン」 ／ 下に「ゾーン解析」「検出結果テーブル」

## 操作手順
1. サイドバーで**タスク**（セグ/追跡/ゾーン）と**モデル・しきい値・対象クラス**を設定。
2. ゾーン解析を使う場合は、トラッキングをON にし、**ゾーン定義(JSON)** を編集（既定例は中央の四角形「ゾーンA」）。
   - 形式：`[{"name":"ゾーンA","polygon":[[0.3,0.3],[0.7,0.3],[0.7,0.9],[0.3,0.9]]}]`
   - 座標は画面比率 0〜1（左上 0,0 / 右下 1,1）。
3. **mp4 をアップロード** → **▶ Run 解析** を押す。
4. 進捗バー（`解析中… n/total フレーム`）が進む。初回はモデル重みの自動DLで少し待つ。
5. 完了後、結果を確認・ダウンロード。

## 出力・結果の見方
- **映像プレビュー**: bbox＋クラス名＋信頼度（追跡ONなら `#ID`、セグONならマスク、ゾーンONなら黄枠）。
- **結果ペイン（右）**:
  - 総検出数 / ユニークID数 / 処理フレーム
  - クラス別（延べ＝のべ検出数、最大同時＝1フレーム内の最大数）
  - **⬇ CSV / ⬇ JSON / ⬇ 注釈付き動画** ダウンロード
  - **📝 NL要約（Claude）**: 検出・ゾーン結果を自然言語で要約（要 `ANTHROPIC_API_KEY`）
- **ゾーン解析（下）**: ゾーン別 通過ID数／侵入回数／最大同時／合計・最大滞留(秒)、および ID別滞留時間。
- **検出結果テーブル**: frame / time_sec / class / conf / bbox / tracker_id。

## つまずきやすい点（トラブルシュート）
| 症状 | 対処 |
|---|---|
| ▶ Run が押せない | 先に mp4 をアップロードする |
| 「対象クラスを1つ以上選ぶか…」警告 | クラスを選ぶ or「全クラス」をON |
| 「ゾーン定義(JSON)の解析に失敗」 | JSON 構文と `polygon` が `[[x,y],...]`（0〜1）か確認 |
| 「モデルの読み込みに失敗」 | 初回はネット接続（重み自動DL）を確認 |
| 動画がプレビュー再生できない | 「⬇ 注釈付き動画」でDLして再生（コーデック依存） |
| ゾーン解析のチェックが押せない | トラッキングをONにする（ゾーンはID前提） |
| 処理が遅い/重い | フレーム間引きを上げる、モデルを n/s に、対象クラスを絞る |
| NL要約でエラー | `.env` の `ANTHROPIC_API_KEY` を設定 |
